# Synapsa mvp

## Crete the branch for components or changes

## Run frontend:

```
frontend/
├── app/        # Aplicação principal (RoleSelect, Home, etc.)
└── landing/    # Landing page (Navbar, Hero, FinalCTA...)
```
## Requisites:

```
Node 18+ (ou LTS atual)

npm (ou yarn/pnpm, se preferir)
```

## Comand Line:

```
cd frontend/landing
npm install
npm run dev
# open http://localhost:8081
```

```
cd frontend/app
npm install
npm run dev
# open http://localhost:8080
```
